Welcome to CleverDoc's documentation!
=====================================

CleverDoc is a library for processing and de-identification of clinical data.
It is designed to be used in a variety of contexts, from simple data processing to complex machine learning pipelines.

CleverDoc is build on top of Spark and provides a set of tools to process clinical data in a distributed way.

CleverDoc is developing by `ApicomPro <http://apicom.pro>`_ team.
